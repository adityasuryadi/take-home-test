
# Take Home Test

## Installation

Install my-project with npm

```bash
  npm install my-project
  cd my-project
```
    
## Run Locally

Clone the project

```bash
  git clone https://github.com/adityasuryadi/take-home-test.git
```

Go to the project directory

```bash
  cd take-home-test
```

Install dependencies

```bash
  composer install
```

Run Migration

```bash
  php artisan migrate
```

Generate JWT Secret

```bash 
  php artisan jwt:secret
```

setting .env

```bash 
  cp .env.example .env
```


Start the server

```bash
  php artisan serve
```
